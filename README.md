# 📟 RetroVane

**RetroVane** es un dashboard de clima con estética retro-futurista inspirado en las terminales de fósforo verde de los años 70 y 80. Este proyecto combina tecnología moderna con una interfaz de usuario nostálgica (estilo CRT).


## 🚀 Características
- **Interfaz de Terminal:** Estética CRT con efecto de parpadeo (flicker) y líneas de escaneo (scanlines).
- **Datos en Tiempo Real:** Integración con la API de OpenWeatherMap.
- **Visualización ASCII:** Iconos climáticos y tipografía representados en estilo ASCII/Pixel art.
- **Responsive:** Adaptado para verse como una terminal tanto en escritorio como en dispositivos móviles.

## 🛠️ Stack Tecnológico
- **Frontend:** React.js / Vite
- **Estilos:** Tailwind CSS.
- **API:** [OpenWeatherMap API](https://openweathermap.org/)
- **Despliegue:** Vercel / GitHub Pages
